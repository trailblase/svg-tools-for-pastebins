> This READ.me will be brief since this website is severly unoptimised and uses a lot of "duck-tape" to function.

**Trailblase** is the name of the website that offers easy and customisable tools for Pastebins. The following options are available:
1. SVG Generator
2. Gradient Markdown Text
3. SVG Dividers
4. Fake Spotify Widget (no login required)
5. Circle Widget
6. Experimental Page (includes all features that are still not polished enough)

***
My biggest aim right now is to severly cut from the bloated code and use a framework to improve functionality.
